# Experiments

MLflow run summaries and experiment metadata.

## Files

| File | Description |
|---|---|
| `runs_summary.md` | Auto-generated top-N run comparison (generated by `make report`) |

## Generating a Run Report

```bash
make report
```

This queries the local MLflow tracking store and writes `experiments/runs_summary.md`
with the top 10 runs sorted by accuracy.

## Viewing in MLflow UI

```bash
make mlflow-ui
# then open http://localhost:5000
```
